🌍✈ Tour & Travel Management System

A simple and functional Tour & Travel Management System developed using Python, HTML, CSS, and MySQL. This project allows users to explore travel packages, register/login, and book trips easily.

---

🚀 Features

- User Registration & Login System
- Explore Travel Destinations
- Booking Functionality
- Contact Form
- Database Integration (MySQL)

---

🛠️ Technologies Used

- Python (Backend)
- HTML, CSS (Frontend)
- MySQL (Database)

---

📂 Project Structure

- app.py (Main application)
- templates/ (HTML files)
- static/ (CSS, images)
- Other Python modules for booking, login, register, etc.

---

▶️ How to Run

1. Install Python
2. Install required libraries (Flask, MySQL connector)
3. Setup MySQL database
4. Run `app.py`
5. Open browser and go to: http://127.0.0.1:5000/
